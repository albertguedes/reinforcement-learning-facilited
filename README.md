# Reinforcement Learning Facilitated

Minimal, readable implementations of **reinforcement learning** algorithms in Python.

## Algorithms

| Folder | Description |
|--------|-------------|
| `stochastic_discrete_states_actions` | Basic stochastic RL with discrete state/action spaces |
| `deterministic_discrete_states_actions` | Deterministic policy RL |
| `stochastic_continous_state_discrete_actions` | Continuous state, discrete actions |
| `deep_qlearning` | Deep Q-Network (DQN) implementation |

## Tech Stack

- Python
- NumPy
- PyTorch (for DQN)

## Usage

```bash
git clone https://github.com/albertguedes/reinforcement-learning-facilited.git
cd reinforcement-learning-facilited
python <algorithm-folder>/main.py
```
